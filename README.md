# Blog Platform

[English version → README.en.md](README.en.md) | [Русская версия → README.ru.md](README.ru.md)

Blog Platform is a multi-project solution for publishing articles and media content. It includes an ASP.NET Core backend, a Vite-based web frontend, and supporting libraries (plus an optional WPF client).

- Backend: `BlogContent.WebAPI`
- Frontend: `blogplatform-frontend`
- Shared layers: `BlogContent.Core`, `BlogContent.Data`, `BlogContent.Services`

For setup and run instructions, choose your language above.
